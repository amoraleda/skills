# Speech Analysis Reference

Detailed patterns and benchmarks for speech coaching analysis.

## Filler Word Categories

### Primary Fillers (High Impact)
| Filler | Common In | Notes |
|--------|-----------|-------|
| um, uh | Universal | Most noticeable, reduce first |
| like | Casual speech | Often unconscious |
| you know | Conversational | Seeking validation |
| basically | Technical | Oversimplification signal |
| actually | Corrections | Can sound defensive |

### Secondary Fillers (Medium Impact)
| Filler | Common In | Notes |
|--------|-----------|-------|
| so | Transitions | Acceptable in moderation |
| okay | Confirmations | Fine when genuine |
| right | Seeking agreement | Can be engaging |
| I mean | Clarifications | Shows self-correction |

### Language-Specific Patterns

**Spanish speakers** often transfer:
- "eh" → "uh"
- "bueno" → "well", "okay"
- "pues" → "so"
- "vale" → "okay"

**These are normal** and don't impede understanding. Focus on frequency, not elimination.

## Sentence Completion Patterns

### Types of Incomplete Sentences

1. **Self-interruption**: Speaker changes direction mid-thought
   > "The thing is that we— actually, let me explain it differently"

2. **External interruption**: Another speaker cuts in
   > "So the implementation would— [Other: What about testing?]"

3. **Restart/Stutter**: Same word repeated while thinking
   > "The the the enterprise version"

4. **Abandoned thought**: Sentence trails off
   > "And then we could maybe..."

### Scoring Guide

| Completion Rate | Assessment |
|-----------------|------------|
| 90%+ | Excellent |
| 80-89% | Good |
| 70-79% | Needs attention |
| <70% | Priority focus area |

## Structure Assessment

### Signpost Phrases (Positive Indicators)

| Purpose | Examples |
|---------|----------|
| Opening | "I'll cover three points today" |
| Sequencing | "First... Second... Finally..." |
| Transitioning | "Moving on to...", "Now let's look at..." |
| Emphasizing | "The key point here is...", "Most importantly..." |
| Summarizing | "To summarize...", "In short..." |
| Returning | "Coming back to...", "As I mentioned..." |

### Structure Score Rubric

| Score | Description |
|-------|-------------|
| 5 | Clear intro, logical flow, smooth transitions, strong conclusion |
| 4 | Good structure with minor jumps, uses signposts |
| 3 | Adequate structure, some topic jumping, few signposts |
| 2 | Disorganized, frequent topic changes, hard to follow |
| 1 | No discernible structure, stream of consciousness |

## Engagement Indicators

### Positive Engagement
- Using names: "Victor, that's a great point"
- Asking questions: "Does that make sense?"
- Acknowledging contributions: "As Maria mentioned earlier"
- Inviting participation: "What do you think about..."

### Negative Patterns
- Monologuing without pauses
- Ignoring questions
- Not acknowledging others' input
- Talking over others

## Benchmarks by Context

### Formal Presentation
- Filler rate: <3 per minute (ideal)
- Structure score: 4-5 expected
- Completion rate: 85%+

### Team Meeting
- Filler rate: <5 per minute (acceptable)
- Structure score: 3+ acceptable
- Completion rate: 75%+ (interruptions expected)

### Casual Discussion
- Filler rate: <8 per minute (normal)
- Structure score: 2+ acceptable
- Completion rate: 70%+ (natural conversation)

## Progress Tracking Metrics

### Key Performance Indicators

1. **Filler Density**: Fillers per minute of speaking
2. **Completion Rate**: % of sentences finished
3. **Structure Score**: 1-5 scale
4. **Engagement Score**: Based on interaction patterns
5. **Clarity Score**: Based on precision of language

### Improvement Timeline Expectations

| Metric | 2 Weeks | 1 Month | 3 Months |
|--------|---------|---------|----------|
| Filler reduction | 20-30% | 40-50% | 60%+ |
| Completion improvement | 5-10% | 15-20% | 25%+ |
| Structure improvement | +0.5 | +1.0 | +1.5 |

## Coaching Tone Guidelines

### Do
- Start with genuine strengths
- Use specific examples from transcript
- Provide actionable, concrete advice
- Acknowledge context (non-native speaker, informal setting)
- Frame improvements as refinements, not fixes

### Don't
- Be overly critical
- Focus only on negatives
- Give vague advice ("speak better")
- Compare to unrealistic standards
- Ignore cultural/linguistic context
